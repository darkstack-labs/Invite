# THE WORST BATCH SIGNING OFF

An exclusive digital invitation website for the **Batch Farewell Party 2026**.

This project is a modern web application built using **React and Vite**, designed to deliver an interactive invitation experience with smooth animations, responsive layouts, and Progressive Web App support.

---

## Project Overview

This site serves as the official invitation platform for **#THEWORSTBATCHSIGNINGOFF**, providing guests with information about the event, schedule, dress code, menu, and RSVP system.

---

## Tech Stack

The project is built with the following technologies:

* **Vite** – fast frontend build tool
* **React** – component-based UI library
* **TypeScript** – type-safe JavaScript
* **Tailwind CSS** – utility-first styling
* **shadcn/ui** – modern UI components
* **PWA Support** – installable web application

---

## Running the Project Locally

Make sure **Node.js** and **npm** are installed.

1. Install dependencies

```
npm install
```

2. Start the development server

```
npm run dev
```

3. Open the site in your browser

```
http://localhost:8080
```

---

## Building for Production

To generate the production build:

```
npm run build
```

Preview the production build:

```
npm run preview
```
